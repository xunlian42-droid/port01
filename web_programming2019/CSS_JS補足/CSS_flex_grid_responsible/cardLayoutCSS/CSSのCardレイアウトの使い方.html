カードレイアウトのCSSサンプルと、その解説をします。カードレイアウトは、画像、タイトル、本文などの情報をひとまとまりのブロック（カード）として表示する、ウェブデザインでよく使われる手法です。

カードレイアウトのHTML構造

まずは、カードの基本となるHTMLを準備します。ここでは、div要素を使って、カード全体を表す親要素と、その中に入る子要素を構成します。

<div class="card-container">
  <div class="card">
    <img src="images/item1.png" alt="カードの画像">
    <div class="card-content">
      <h3>カードのタイトル</h3>
      <p>ここにカードの詳細な説明文が入ります。この文章はカードの内容を簡潔に伝えます。</p>
      <a href="#" class="read-more">もっと見る</a>
    </div>
  </div>
  <div class="card">
    <img src="images/item2.png" alt="カードの画像">
    <div class="card-content">
      <h3>カードのタイトル</h3>
      <p>ここにカードの詳細な説明文が入ります。この文章はカードの内容を簡潔に伝えます。</p>
      <a href="#" class="read-more">もっと見る</a>
    </div>
  </div>
</div>

CSSによるカードレイアウトの解説

次に、上記のHTMLにスタイルを適用します。このサンプルでは、Flexboxを使って複数のカードを簡単に並べる方法も解説します。

/* 全体を囲むコンテナのスタイル */
.card-container {
  display: flex; /* 子要素（カード）をFlexboxで並べる */
  flex-wrap: wrap; /* 画面が狭くなったら折り返す */
  gap: 20px; /* カードとカードの間の余白 */
  padding: 20px;
  justify-content: center; /* カードを中央寄せにする */
}

/* 個々のカードのスタイル */
.card {
  width: 300px; /* カードの幅を固定 */
  border: 1px solid #ddd; /* 薄い灰色の枠線 */
  border-radius: 8px; /* 角を丸くする */
  overflow: hidden; /* カードの角からはみ出る要素を隠す */
  box-shadow: 0 4px 8px rgba(0, 0, 0, 0.1); /* ほんのり影をつける */
  transition: transform 0.3s ease; /* マウスホバー時のアニメーション */
  background-color: white; /* 背景色 */
}

/* カードにマウスを乗せたときのスタイル */
.card:hover {
  transform: translateY(-5px); /* 上に少し移動させる */
  box-shadow: 0 8px 16px rgba(0, 0, 0, 0.2); /* 影を濃くする */
}

/* カード内の画像スタイル */
.card img {
  width: 100%; /* 画像の幅をカードの幅に合わせる */
  height: 200px; /* 画像の高さを固定 */
  object-fit: cover; /* 画像が歪まないようにフィットさせる */
}

/* カード内のコンテンツ（タイトル、本文など）のスタイル */
.card-content {
  padding: 15px;
}

.card-content h3 {
  margin-top: 0; /* h3タグのデフォルトの余白を調整 */
}

.card-content p {
  color: #555; /* 文字色を濃いグレーに設定 */
  font-size: 0.9em; /* フォントサイズを親要素の90%に設定 */
  line-height: 1.6; /* 行の高さを文字サイズの1.6倍に設定し、読みやすくする */
}

/* 「もっと見る」ボタンのスタイル */
.read-more {
  display: inline-block; /* インライン要素をブロック要素のように扱い、幅や高さを設定可能にする */
  margin-top: 10px; /* 上側の外側余白を10pxに設定 */
  padding: 8px 15px; /* 内側の余白を上下8px、左右15pxに設定 */
  background-color: #007BFF; /* 背景色を青色に設定 */
  color: white; /* 文字色を白に設定 */
  text-decoration: none; /* テキストの下線を非表示にする */
  border-radius: 5px; /* 角を丸くする */
  font-size: 0.9em; /* フォントサイズを親要素の90%に設定 */
}

.read-more:hover {
  background-color: #0056b3; /* マウスを乗せた時の背景色を少し濃い青色に変更 */
}

各プロパティの解説

  • .card-container:

    • display: flex;とflex-wrap: wrap;を組み合わせることで、画面幅に合わせてカードが自動的に折り返され、レスポンシブなレイアウトが簡単に作れます。
    • gap: 20px;は、Flexboxの便利な機能で、アイテム間の余白を一括で指定できます。

  • .card:

    • width: 300px;でカードの幅を固定しています。
    • border-radius: 8px;で角を丸くし、box-shadowで立体感を演出しています。
    • transitionは、マウスを乗せたときのアニメーションを滑らかにするための設定です。

  • .card img:

    • width: 100%;とheight: 200px;で画像サイズを統一し、object-fit: cover;で画像が指定サイズにぴったり合うように調整しています。これにより、画像の縦横比を保ったまま、カード全体が崩れないようにできます。

このサンプルを参考に、画像のURLやテキストを自由に変更して、オリジナルのカードを作成してみてください。
