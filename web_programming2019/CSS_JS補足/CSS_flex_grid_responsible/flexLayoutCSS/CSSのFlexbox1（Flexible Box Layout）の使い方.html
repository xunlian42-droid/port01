HTMLとCSSのFlexbox（Flexible Box Layout）の使い方を、簡単なサンプルを使って説明します。

Flexboxとは？

Flexboxは、ウェブページの要素を柔軟に配置するためのCSSのレイアウト機能です。特に、要素を横や縦に並べたり、中央に寄せたり、均等に配置したりする際に非常に便利です。以前は複雑なCSSやHTMLの構成が必要でしたが、Flexboxを使うとシンプルに実現できます。

Flexboxのレイアウトは、**親要素（Flexコンテナ）と子要素（Flexアイテム）**という2つの要素で構成されます。

  1. Flexコンテナ: 子要素を管理する親要素です。
  2. Flexアイテム: コンテナ内で配置される子要素です。

基本的な使い方とサンプル

以下に、基本的なHTMLとCSSのサンプルを示します。

1. HTMLの準備

まず、Flexboxで配置したい子要素を、親要素で囲みます。この親要素がFlexコンテナになります。

<div class="flex-container">
  <div class="flex-item">アイテム1</div>
  <div class="flex-item">アイテム2</div>
  <div class="flex-item">アイテム3</div>
</div>

2. CSSでFlexboxを有効にする

Flexboxを使うには、親要素（.flex-container）にdisplay: flex;というCSSプロパティを指定するだけです。これだけで、子要素は自動的に横並びになります。

.flex-container {
  /* これがFlexboxを有効にする最も重要な設定です */
  display: flex; 

  /* 見やすくするために背景色や余白を設定 */
  background-color: #f0f0f0;
  padding: 10px;
  border: 1px solid #ccc;
}

.flex-item {
  background-color: #4CAF50;
  color: white;
  padding: 20px;
  margin: 5px;
  font-size: 1.2em;
  text-align: center;
}

このCSSを適用すると、div.flex-itemが横一列に並んで表示されます。

よく使うプロパティ

Flexboxには、子要素の配置を細かく制御するための便利なプロパティがいくつかあります。これらは主に**親要素（Flexコンテナ）**に指定します。

1. justify-content (水平方向の配置)

子要素を水平方向にどのように配置するかを決めます。

  • justify-content: center;
    • 中央寄せにします。
  • justify-content: space-between;
    • 両端にアイテムを配置し、その間の余白を均等にします。
  • justify-content: space-around;
    • アイテムの周りの余白を均等にします。

2. align-items (垂直方向の配置)

子要素を垂直方向にどのように配置するかを決めます。

  • align-items: center;
    • 中央寄せにします。
  • align-items: flex-start;
    • 上端に寄せます。
  • align-items: flex-end;
    • 下端に寄せます。

3. flex-direction (配置の向き)

子要素を並べる方向を決めます。

  • flex-direction: row;
    • **横方向（デフォルト）**に並べます。
  • flex-direction: column;
    • 縦方向に並べます。

応用例：中央寄せ

justify-contentとalign-itemsを両方使うことで、アイテムをコンテナの真ん中に完璧に中央寄せできます。

.flex-container {
  display: flex;
  justify-content: center; /* 水平方向の中央寄せ */
  align-items: center; /* 垂直方向の中央寄せ */
  height: 200px; /* 親要素の高さを指定しないと垂直方向の中央寄せは機能しません */
}

このCSSを使えば、子要素がコンテナのちょうど真ん中に配置されます。
