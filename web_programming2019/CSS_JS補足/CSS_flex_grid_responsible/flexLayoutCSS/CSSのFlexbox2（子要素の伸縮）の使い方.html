ステップ2：アイテムのサイズと並び順をコントロールする

Flexboxは、子要素（Flexアイテム）のサイズや並び順を、CSSだけで簡単に変更することができます。

1. flex-grow と flex-shrink (アイテムの伸縮)

親要素に余白がある場合にアイテムを広げたり、逆にスペースが足りない場合に縮めたりする設定です。

  • flex-grow: 1;
    • 余ったスペースを均等に埋める設定です。複数のアイテムにflex-grow: 1;を指定すると、余白を均等に分け合って広がります。
  • flex-shrink: 1;
    • スペースが足りない場合に縮む設定です。デフォルトで1が指定されており、通常はこのままで問題ありません。

サンプルコード

.flex-item-grow {
  flex-grow: 1; /* 親要素の余白を埋めて広がる */
}

.flex-item-no-grow {
  flex-grow: 0; /* 広がらない */
}

上の例では、flex-item-growを持つアイテムだけが、残りの余白を埋めるように幅が広がります。

2. order (アイテムの並び順)

orderプロパティを使うと、HTMLの記述順に関係なく、アイテムを好きな順番に並べ替えることができます。

  • order: 1;, order: 2;…
    • orderの値が小さい順に左から並びます。デフォルトの値は0です。
    • 例えば、3番目のアイテムを最初に表示したい場合は、そのアイテムにorder: -1;を設定すれば実現できます。

サンプルコード

<div class="flex-container">
  <div class="flex-item item-a">アイテムA (order: 0)</div>
  <div class="flex-item item-b">アイテムB (order: 2)</div>
  <div class="flex-item item-c">アイテムC (order: 1)</div>
</div>

.item-a {
  order: 0;
}
.item-b {
  order: 2; /* 2番目に表示される */
}
.item-c {
  order: 1; /* 1番目に表示される */
}

このCSSを適用すると、「アイテムA」「アイテムC」「アイテムB」の順に表示されます。

Flexboxは、これらのプロパティを組み合わせることで、複雑なレイアウトもシンプルに作成できるのが大きな強みです。まずはdisplay: flex;と、今回ご紹介したプロパティを試してみてください。
