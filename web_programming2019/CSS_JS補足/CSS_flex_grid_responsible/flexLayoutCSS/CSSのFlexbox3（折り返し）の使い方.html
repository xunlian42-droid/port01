承知いたしました。Flexboxのステップ1とステップ2に続き、さらに高度な機能である折り返しと個別のアイテム調整について、ステップ3として解説します。

ステップ3：アイテムの折り返しと個別調整

Flexboxでは、画面幅が足りないときにアイテムを自動的に折り返したり、特定のアイテムだけを個別にカスタマイズしたりできます。

1. flex-wrap (アイテムの折り返し)

親要素にdisplay: flex;を設定すると、デフォルトではアイテムは1行に収まるように縮小されます。しかし、flex-wrapプロパティを使うことで、アイテムを複数行に折り返すことができます。

  • flex-wrap: nowrap;
    • 折り返さない（デフォルト設定）。アイテムは1行に収まるように縮小されます。
  • flex-wrap: wrap;
    • アイテムを複数行に折り返します。画面幅が足りなくなると、自動的に次の行に送られます。これは、レスポンシブデザインで特に役立ちます。

サンプルコード

.flex-container {
  display: flex;
  flex-wrap: wrap; /* これを設定するとアイテムが折り返されます */
  gap: 10px; /* アイテム間の余白 */
  width: 300px; /* 親要素の幅を意図的に狭く設定 */
}

.flex-item {
  width: 120px; /* アイテムの幅を固定 */
  height: 50px;
  background-color: #3498db;
  color: white;
  text-align: center;
  line-height: 50px;
}

このコードでは、親要素の幅が300pxなので、幅120pxのアイテムは2つしか横に並べられません。flex-wrap: wrap;のおかげで、3つ目のアイテムは自動的に下の行に折り返されます。

2. align-self (特定のアイテムだけを調整)

align-itemsはコンテナ内のすべてのアイテムを垂直方向に調整しましたが、**align-self**を使うと、特定のFlexアイテムだけを個別に調整できます。

  • align-self: flex-start;
    • そのアイテムだけを上端に寄せます。
  • align-self: center;
    • そのアイテムだけを中央寄せにします。
  • align-self: flex-end;
    • そのアイテムだけを下端に寄せます。

サンプルコード

.flex-container {
  display: flex;
  align-items: center; /* すべてのアイテムを垂直方向の中央に寄せる */
  height: 150px;
  background-color: #f0f0f0;
}

.flex-item {
  width: 80px;
  background-color: #e74c3c;
  color: white;
  text-align: center;
  padding: 15px;
  margin: 5px;
}

.special-item {
  align-self: flex-end; /* このアイテムだけを下端に寄せる */
  background-color: #2ecc71;
}

このCSSを適用すると、すべてのアイテムは中央に配置されますが、special-itemクラスを持つアイテムだけが下端に配置されます。

これらのプロパティを組み合わせることで、より複雑で柔軟なレイアウトを簡単に作成できます。Flexboxの強力な機能の多くは、この3つのステップでほぼカバーできるので、まずはこれらの使い方をマスターしてみてください。
