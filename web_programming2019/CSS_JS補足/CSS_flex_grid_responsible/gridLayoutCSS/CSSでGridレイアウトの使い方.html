CSS Gridのサンプルコードについて、初心者の方にも分かりやすいように解説します。

このコードは、ウェブページで要素を格子状に並べるためのCSS Gridという機能を使っています。例えるなら、エクセル（Excel）の表のようなものを作るイメージです。 Flexレイアウトが１次元のレイアウトを得意とするのに対して、Gridレイアウトが２次元のレイアウトを得意とします。

1. grid-container（親要素）の設定

まず、grid-containerというクラスが付いた<div>が、グリッドの全体を管理する「親要素」となります。この親要素に、グリッドを動かすための設定を書いていきます。

  • display: grid;: 👨‍💻これが一番重要！この1行で、この要素がCSS Gridとして動き出すことを宣言しています。この要素の中に含まれる子要素（.grid-item）は、自動的にグリッドのセルに配置されるようになります。

  • grid-template-columns: 1fr 1fr 1fr;: 🔢横方向の列（カラム）をどう並べるかを決めています。

    • 1frは「利用できる空間を1つ分使う」という意味です。
    • 1fr 1fr 1frと書くことで、3つの列を均等な幅で作りなさい、という指示になります。

  • grid-template-rows: 100px 100px;: 📏縦方向の行（ロウ）をどう並べるかを決めています。

    • 100pxと書くことで、高さを100ピクセルに設定しています。
    • 100px 100pxと書くと、高さ100pxの行を2つ作りなさい、という指示になります。

  • gap: 10px;: 🖼️それぞれのマス（セル）の間隔を調整しています。

    • 10pxは「10ピクセル分の隙間」を空けるという意味です。
    • 行と列の間すべてに、10pxの隙間ができます。

これらの設定により、「横3列、縦2行」 のグリッドが完成します。

2. grid-item（子要素）の設定

次に、.grid-containerの中にあるgrid-itemというクラスが付いた<div>が、グリッドの中に配置される「子要素」です。これらの要素は、自動的にグリッドのマス目の中に順番に入っていきます。

  • background-color: #4CAF50;: 背景色を緑色にしています。
  • color: white;: 文字の色を白色にしています。
  • text-align: center;: 文字をセルの中央に配置しています。
  • line-height: 100px;: これは少し特別な設定です。line-heightをgrid-template-rowsで指定した高さと同じ100pxにすることで、文字が垂直方向にも中央に揃います。
  • font-size: 2em;: フォントのサイズを大きくして、見やすくしています。

このHTMLとCSSを組み合わせると、下のようなレイアウトが完成します。

grid-containerに設定したgapのおかげで、それぞれのセルの間に10pxの隙間ができていますね。

CSS Gridは、このように親要素に設定を集中させるだけで、中の要素のレイアウトをとても簡単に組むことができる便利な機能です。
